Documentation ApiRTC

Lexique des objets

UserAgent

On instancie un UserAgent avec une clef d'API fournie par Apizee et associée à votre compte sur apizee.com : l'apiKey


userAgent = new UserAgent({
    uri: 'apzkey:' + this.apiKey
    })

C'est un point d'entrée pour enregistrer un participant auprès des serveurs d'Apizee

L'enregistrement permet d'obtenir une Session qui permettra d'utiliser les fonctionnalités de conference/conversation

Session

On obtient une instance de Session en retour (Promise) de la méthode UserAgent.register()


userAgent.register().then(session => {
        // use session
    };

Conversation

On obtient une instance de Conversation en retour de la méthode Session.getOrCreateConversation(name)

note : ceci est bizarre pour joindre ume conversation c'est conversation.join() sémantiquement on aurai

L'appel à la méthode Conversation.join() permet au participant de rejoindre la conversation
